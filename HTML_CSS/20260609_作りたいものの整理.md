# 作りたいものメモ
- 映画の鑑賞ログ

## 開発の流れ
- 要件定義 ← 済
- 画面設計 ← 済
- DB設計・作成 ← 出来そう
- バックエンド実装（Java / Spring Boot）←Javaは出来そう
- フロントエンド実装（HTML / CSS）←レスポンシブが課題
- 結合・テスト

## 考えなくちゃいけないこと
- 要件定義
- 画面設計（どんな画面が必要か）
- 環境構築
- 実装

### 最低限必要な機能
CRUDがあれば動く。  
映画の登録、表示、更新、削除

### DBの管理で最低限必要なもの
- 主キー(INT型)
- 見た日付（DATE型）
- タイトル(VARCHAR型)

後々追加出来たらいいな～  
視聴した媒体(VARCHAR型)、ジャンル(VARCHAR型)

### 必要な画面設計
- 入力画面(C/U)
- 入力結果画面(R)
- 一覧画面(R)
- 削除画面(D)

### 環境構築

| 役割 | ツール | 理由 |
|------|--------|------|
| IDE | **VSCode** | 拡張機能入れる |
| フレームワーク | **Spring Boot** | そのまま |
| DB | **MySQL** | もう入ってる |
| ビルドツール | **Maven** | そのまま |
| バージョン管理 | **Git** | もう使ってる |

| 拡張機能 | 用途 |
|----------|------|
| **Extension Pack for Java** | Java開発に必要なものが一括で入る |
| **Spring Boot Extension Pack** | Spring Boot開発に必要なものが一括で入る |
