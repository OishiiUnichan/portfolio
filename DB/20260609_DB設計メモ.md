# DB設計メモ

## 環境
- Java：11
- フレームワーク：Spring Boot（必要な部分だけ学習して使う）
- DB：MySQL（導入済み）
- IDE：VSCode（環境構築済み）

## テーブル設計

### テーブル名：`movies`

| 列名 | 型 | 制約 | 備考 |
|------|----|------|------|
| id | INT | PRIMARY KEY / AUTO_INCREMENT | |
| watch_date | DATE | NOT NULL | 見た日付 |
| title | VARCHAR(255) | NOT NULL | タイトル |
| media | VARCHAR(255) | | 視聴媒体（任意） |
| genre | VARCHAR(255) | | ジャンル（任意） |

## 決定事項メモ
- 列名は英語（エラー防止のため）
- `id` は自動採番（AUTO_INCREMENT）
- `title` と `watch_date` は必須（NOT NULL）
- `media` と `genre` は最初から追加、入力は任意
