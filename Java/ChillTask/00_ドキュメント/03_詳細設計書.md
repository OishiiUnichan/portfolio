# 03_詳細設計

## 3.1. クラス設計（Javaクラス・構造）

### `Task` （単一タスクモデル）

* **フィールド:**
* `private int id`
* `private String title`
* `private boolean isCompleted`
* `private LocalDate createdAt`


* **メソッド:**
* ゲッター / セッター
* `public void toggleComplete()` (完了状態の反転処理)



### `TaskManager` (タスク管理・ガチャ条件判定ロジック)

* **フィールド:**
* `private List<Task> taskList`
* `private int completedCountForGacha`
* `private boolean isDemoMode`


* **メソッド:**
* `public void addTask(String title)`
* `public void deleteTask(int id)`
* `public boolean completeTask(int id)` (タスク完了処理。ガチャ条件達成時に `true` を返す)
* `public void resetGachaCounter()`



### `Furniture` (家具基底 / デコ家具モデル)

* **フィールド:**
* `protected String id` (例: "DECO_PLANT_01")
* `protected String name`
* `protected String imagePath`
* `protected String areaId` (例: "DESK_ZONE")
* `protected boolean isAcquired`



### `CoreFurniture` (`Furniture` を継承 / 基本家具モデル)

* **フィールド:**
* `private int level` (初期値: 1)
* `private int maxLevel`


* **メソッド:**
* `public void upgrade()` (レベルを+1し、画像パスを更新)



### `RoomManager` (部屋の状態管理・抽象ロジック)

* **フィールド:**
* `private Map<String, List<Furniture>> roomLayout` (エリアIDがキーの配置マップ)
* `private List<Furniture> gachaPool` (未獲得・未カンスト家具の抽選プール)


* **Methods:**
* `public void initRoom()` (初期家具を配置した部屋のビルド)
* `public Furniture rollGacha()` (プールからランダムに1つ抽出し、状態を更新して返す)
* `public Map<String, List<String>> getRoomImagesJson()` (フロントに渡す画像パスリストの生成)



### `AvatarManager` (アバター行動制御)

* **フィールド:**
* `private String currentActionId` (例: "SITTING", "SLEEPING")


* **メソッド:**
* `public String determineAction(int hour)` (時間帯から行動IDを決定)
