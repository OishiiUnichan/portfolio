# 00_概要

* **プロジェクト名:**  
  ChillTask (チルタスク)
* **コンセプト:**  
 「日々の作業の積み重ねが、自分だけのチルい空間を豊かにしていく」
* **概要:**  
    ユーザーが日常のタスクを登録・管理し、それをクリアすることで、Blenderで自作したアイソメトリック（クォータービュー）の家具アセットをランダムに獲得。  
  自分だけの部屋を全自動で充実させていく、ゲーミフィケーションを取り入れたWeb型進捗管理アプリ。  
* **技術選定理由:**  
    学校で習った基礎の振り返りのため。
* **非機能要件:**  
    一旦後回し。レスポンシブ対応ぐらいしか今のところは考えてない。
* **制作背景:**  
  - 視覚で見た時に面白い物を作りたかった
  - blender学習中なので折角だったら作ったものを使いたい
  - タスク管理アプリは色々使ってるが長続きする要素がなくて使わなくなる
  - 基本のCRUDの練習にタスク管理がちょうどよかった
  - 自分の好きなものを詰め込んだら楽しそうと思った
    - 好きなものとは…
      - サンドボックスゲーム（マイクラ、どうぶつの森）
      - ハウジング、インテリア要素
      - チルゲー(Chill With You,Gosh etc...)
  

  自分の好きなものの要素を満たせるものはそれなりにあるけど全部は満たしてない。  
    - こだわり  
      - 自動化：家具配置のリソースを割かない（自由度が高すぎるとタスク管理から外れて本末転倒）  
      - キャラクター：アバターを人間と動物で選択できる（人間以外も選べる方が楽しそう）
